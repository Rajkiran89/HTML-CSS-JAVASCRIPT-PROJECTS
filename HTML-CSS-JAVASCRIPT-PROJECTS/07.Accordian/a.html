• HTML stands for Hypertext Markup Language and is the standard markup language used to create web pages. It's the most basic building block of the web and is used to structure web pages, add visual formatting, and make them accessible to users.

  • CSS stands for Cascading style sheets. It describes to the user how to display HTML elements on the screen in a proper format. CSS is the language that is used to style HTML documents.In simple words, cascading style sheets are a language used to simplify the process of making a webpage.

  • JavaScript (JS) is a programming language that developers use to create interactive web pages

  • React is a JavaScript library that's used to build user interfaces (UIs) for web and native platforms
